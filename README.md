# Pixel Manipulation for Image Encryption

## Description
A simple image encryption and decryption tool developed using Python and Pillow. The project encrypts an image by modifying pixel values using a secret key and restores it through decryption.

## Technologies Used
- Python
- Pillow (PIL)

## Features
- Encrypt images using pixel manipulation
- Decrypt images back to original form
- Simple key-based encryption

## How to Run

1. Install Pillow
   pip install pillow

2. Run Encryption
   python image_encrypt.py

3. Run Decryption
   python image_decrypt.py
